VeVaL Airport Transfers
Welcome to the repository for VeVaL Airport Transfers, a premium private airport transfer service connecting Markham and Richmond Hill to Toronto Pearson Airport (YYZ).

🌟 Overview
This project contains the front-end code for the VeVaL Airport Transfers landing page. It is a single-page, responsive website designed to provide a seamless booking experience for customers seeking reliable, premium transportation. It features a modern UI, details about our flagship vehicle, and direct integration with our secure booking form.

✨ Features
Fully Responsive Design: Optimized for mobile, tablet, and desktop viewing.
Modern UI: Built using Tailwind CSS for a clean, premium, and trustworthy aesthetic.
Integrated Booking: Direct links to the secure Fillout booking portal.
Legal Modals: Vanilla JavaScript-powered interactive modals for the Privacy Policy and Terms of Service, ensuring users don't have to leave the main page to read the fine print.
🚀 Getting Started
This project is built using standard HTML, CSS (via Tailwind CDN), and Vanilla JavaScript. No complex build tools, package managers, or server setups are required to view or edit the site.

Prerequisites
To run this project locally, you just need a modern web browser (Chrome, Safari, Firefox, Edge).

You must also ensure that the following image assets are placed in the same root directory as the index.html file:

VeVaL_logo.png
Chrysler_Pacifica.png
How to Run Locally
Clone this repository or download the files to your local machine.
Verify that your images and index.html are in the exact same folder.
Double-click the index.html file to open and view the website in your default web browser.
🛠️ Built With
HTML5: Clean, semantic markup.
Tailwind CSS: Utility-first CSS framework (loaded rapidly via CDN).
Vanilla JavaScript: Lightweight DOM manipulation for interactive elements (Modals).
